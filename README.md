# Book 1: Kirat Itihas by Iman Singh Chemjong

A repository dedicated to **Kirat Itihas** by **Iman Singh Chemjong**, providing educational resources, study notes, references, summaries, and historical context related to the Kirat civilization and Kirati communities.

## About

This repository serves as a learning resource for students, researchers, and history enthusiasts interested in the history, culture, traditions, and heritage of the Kirat people. It aims to organize references and educational materials related to the work of Iman Singh Chemjong for research and academic purposes.
